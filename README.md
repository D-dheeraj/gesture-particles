# Hand Gesture Particle System ❤️

An interactive 3D particle system using **Three.js** and **MediaPipe Hands** that responds to real-time hand gestures through webcam tracking.

## Features
- ✋ Open hand interaction
- ✌️ Pinch gesture particle repulsion
- ❤️ Double thumbs-up heart mode
- Real-time webcam hand tracking
- 50,000 particle animation
- Heart transformation effect

## Technologies Used
- HTML
- JavaScript
- Three.js
- MediaPipe Hands
- WebGL

## How to Run
1. Download the project
2. Open `index.html`
3. Allow webcam access
4. Try gestures:
   - PINCH ✌️
   - DOUBLE THUMBS UP ❤️
